getSimpleSchemaAllowedValues(field)

Given a field, an array of the allowed values for that field is returned in the form of objects with label/value keys. Such values are taken from the maAllowedValues() function defined in the schema definition. If the maAllowedValues function is not defined, values are taken from the allowedValues field and in this case the label is equal to the value. If allowedValues is also missing, an empty array is returned. This function is used by the templates providing the select and multiselect components.
